# Cyno Haviland WordPress Theme
Theme WordPress chuyên nghiệp cho hệ thống khách sạn Haviland, được phát triển bởi Cyno.

## Yêu cầu hệ thống

- WordPress 6.0 trở lên
- PHP 7.4 trở lên
- MySQL 5.6 trở lên
- Hỗ trợ SSL (HTTPS)
